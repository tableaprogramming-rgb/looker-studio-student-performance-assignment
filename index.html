Task 5: Distribution Analysis (Box Plot)

A Box Plot requires 5 specific values to visualize data spread (Min, 25th Percentile, Median, 75th Percentile, Max). Since our dataset only has one "math score" column, we must manually create 5 fields to satisfy the chart's requirements.

Phase 1: Data Preparation

Before adding the chart, you must create 5 distinct fields in your data source. We will name them starting with Math_ so we clearly know these fields are visualizing math scores (and not Reading or Writing).

  1. In Looker Studio, click Resource > Manage added data sources > Edit.
  2. Click + ADD A FIELD in the top right.
  3. Field 1: Name it Math_Plot_Min. Formula: math score. Click Save, then Back.
  4. Repeatedly click + ADD A FIELD to create 4 more fields with the exact same formula (math score) but different names:
    • Math_Plot_Q1
    • Math_Plot_Median
    • Math_Plot_Q3
    • Math_Plot_Max
  5. Click Done to finish editing the data source.
Phase 2: Configuring the Chart
  1. Add a Box Plot chart to your canvas.
  2. Set Dimension to gender.
  3. Now, drag your 5 new fields into the Metrics section in this exact order:
    1. Min: Drag Math_Plot_Min. Click the pencil icon next to it and change aggregation to MIN.
    2. Lower Quartile: Drag Math_Plot_Q1. Change aggregation to AVERAGE (Note: This is a simulation for this lesson).
    3. Median: Drag Math_Plot_Median. Change aggregation to MEDIAN.
    4. Upper Quartile: Drag Math_Plot_Q3. Change aggregation to AVERAGE (Simulation).
    5. Max: Drag Math_Plot_Max. Change aggregation to MAX.

Note: Because Looker Studio cannot easily calculate percentiles from Sheets data, we are approximating the quartiles using Averages for the purpose of learning the chart layout tool.
