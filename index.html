Scenario: You have been hired as a Junior Data Analyst for the City School District. The Superintendent has provided you with a refined dataset containing test scores and demographic data for 1,000 students. They have asked you to create an interactive dashboard in Looker Studio to help the administration understand which factors‚Äîsuch as parental education or test preparation‚Äîmight be influencing student success.

Welcome to your data visualization training. In this session, we will move beyond simple spreadsheets and create a dynamic report. By the end of this lesson, you will have built a professional dashboard that visualizes the key insights hidden within the rows and columns of your data.

Task 1: The "At a Glance" Metrics

First, we need to establish the baseline. Every good dashboard starts with high-level summary statistics (KPIs).

  1. In the toolbar, click Add a chart and select Scorecard.
  2. Place the scorecard at the top left of your canvas. Ideally, this first card counts the Total Students (Record Count).
  3. Copy and paste this scorecard three times to create a row of four cards.
  4. For the new cards, change the Metric to display:
    • Math Score
    • Reading Score
    • Writing Score
Looker Studio defaults to SUM for numeric fields. You must change the aggregation to AVERAGE for scores.

Warning: Your dataset now contains a student column (ID). Since this is a unique ID (Primary Key), never use SUM or AVERAGE on this field. It is strictly for identifying individual records.

Task 2: Education Impact (Bar Chart)

Does a parent's education level correlate with their child's math score? Let's visualize this using a vertical bar chart.

  1. Add a Column Chart to your canvas.
  2. Set the Dimension to parental level of education.
  3. Set the Metric to math score.
  4. Ensure the metric aggregation is set to Average.
  5. In the Sort options, set it to sort by Metric (Descending) so the highest scores appear on the left.

Task 3: Demographics (Donut Chart)

Let's look at the composition of our student body based on the anonymized "Race/Ethnicity" groups.

  1. Add a Donut Chart (or Pie Chart) to the canvas.
  2. Set the Dimension to race/ethnicity.
  3. Set the Metric to Record Count.
  4. This will automatically calculate the percentage of the total student population for each group.

  • Group D (26.2%)

Task 4: Correlation Analysis (Scatter Plot)

Is there a relationship between reading and writing skills? A scatter plot is perfect for comparing two numerical variables to see if they trend together.

  1. Add a Scatter Chart.
  2. Set the Dimension to student (This is the unique ID from the new Google Sheet. Using this ensures every dot represents one specific student).
  3. Set Metric X to reading score.
  4. Set Metric Y to writing score.
  5. Analyze the pattern. A diagonal line moving up and to the right indicates a strong positive correlation.

Reference: A strong positive correlation (upward diagonal trend).

Task 5: Interactivity (The Filter)

Static reports are useful, but interactive dashboards are powerful. Let's add a filter to allow the Superintendent to segment the data.

  1. In the top menu, click Add a control.
  2. Select Drop-down list.
  3. Place the drop-down at the top right of your dashboard.
  4. Set the Control Field to test preparation course.
  5. Switch to View Mode and try selecting "Completed" vs "None" to see how the charts update dynamically.
